# Система управления библиотекой
### Цель:
Разработать приложение для учета книг в библиотеке

## Учёт книг
- Хранение информации о книгах:
  - Название
  - Автор
  - Год издания
  - Жанр
  - Статус "доступна/взята"
- Добавление, удаление и редактирование книг

## Учет читателей
- Хранение данных о читателях:
  - Имя
  - Фамилия
  - ID
  - Список взятых книг
- Регистрация новых читателей

## Выдача и возврат книг
- Отметка, кто какую книгу взял
- Проверка доступности книги перед выдачей

## Поиск и фильтрация
- Поиск книг по:
  - Названию
  - Автору
  - Жанру
- Вывод списков:
  - Все доступные книги
  - Все взятые книги
